# Quêtes Principales

{% stepper %}
{% step %}
#### [Un Nouveau Départ](un-nouveau-depart.md)

{% hint style="info" %}
Première Quête du Palier 1. Il est demandé d'aller éliminer quelques [Sangliers Corrompus](../../monstres/carte-du-sud/zone-sanglier/sanglier-corrompu.md) et de ramener leurs Peaux à Abraham
{% endhint %}
{% endstep %}

{% step %}
#### [La Vieille Mara](la-vieille-mara.md)

{% hint style="info" %}
Après avoir exploré la [Ville de Départ](../../carte/regions/ville-de-depart.md) et la [Zone des Sangliers](../../carte/regions/zone-sanglier.md), il vous est demandé d'aller dans le village le plus proche ([Hanaka](../../carte/regions/hanaka.md)) pour y entendre les visions d'une vieille femme
{% endhint %}
{% endstep %}

{% step %}
#### [La Corruption](la-corruption.md)

{% hint style="info" %}
Il est temps d'explorer la zone des [Marécages Putrides](../../carte/regions/marecage-putride.md) proche de [Hanaka](../../carte/regions/hanaka.md) afin de découvrir la source des visions de la vieille Mara. Attention aux [Mini Tréants](../../monstres/carte-du-sud/hanaka/mini-treant.md) qui peuple la zone
{% endhint %}
{% endstep %}

{% step %}
#### [Revenir Plus Fort](revenir-plus-fort.md)

{% hint style="info" %}
Quête peu intéressante demandant de monter au Niveau 4. Il est donc temps de se pencher un peu plus sur les [Quêtes Secondaires](../quetes-secondaires/). Nous vous conseillons la Quête d'[Inari](../quetes-secondaires/ville-de-depart/inari.md), extrêmement rentable en cols et en expérience
{% endhint %}
{% endstep %}

{% step %}
#### [L'aventure Commence](laventure-commence.md)

{% hint style="info" %}
Après avoir enfin atteint le niveau 4, votre aventure commence vraiment. Rendez-vous à Mizunari pour faire face aux Nephentes et fabriquez la clé du terrible Donjon Geldorak. Ici, la quête principale se subdivise en deux parties : le sous-donjon Nasgul ([Donjon Squelettes](../../carte/donjons/donjon-squelette.md)) et le [Donjon Geldorak](../../carte/donjons/donjon-geldorak.md)
{% endhint %}
{% endstep %}

{% step %}
#### [Le Ravage des Nephentes](le-ravage-des-nephentes.md)

{% hint style="info" %}
Sauvez Harrold, le mari d'Elma, des Nephentes qui rôdent dans les champs de blé
{% endhint %}
{% endstep %}

{% step %}
#### [Velka](velka.md)

{% hint style="info" %}
Allez voir Velka pour qu'elle vous en apprenne plus sur le Sceau des Anciens
{% endhint %}
{% endstep %}

{% step %}
#### [Les Ruines Maudites](les-ruines-maudites.md)

{% hint style="info" %}
Après avoir écouté les paroles de Velka, cherchez Erik comme elle vous l'a demandé
{% endhint %}
{% endstep %}

{% step %}
#### [Le Donjon des Ruines Maudites](le-donjon-des-ruines-maudites.md)

{% hint style="info" %}
Après avoir parlé à Erik, vous devez plonger dans le sous-donjon Nasgul ([Donjon Squelette](../../carte/donjons/donjon-squelette.md))
{% endhint %}
{% endstep %}

{% step %}
#### [Nasgul: Protecteur Maudit](nasgul-protecteur-maudit.md)

{% hint style="info" %}
Après avoir pénétré le donjon, vous devez ouvrir le chemin vers le maître de ces lieux : le terrifiant [Nasgul](../../monstres/carte-du-sud/ruines-maudites/nasgul.md). Pour cela, vous devez prouver votre valeur en combattant Narax (le Mini-Boss de cet endroit) et en arrachant de son buste son Cœur Putréfié et en récupérant les Tissus Maudits dont se revêtent les [Squelettes Mages](../../monstres/carte-du-sud/ruines-maudites/squelette-mage.md)
{% endhint %}
{% endstep %}

{% step %}
#### [Donjon: Mine de Geldorak](donjon-mine-de-geldorak.md)

{% hint style="info" %}
Une fois le Premier sous-donjon fait, le premier vrai donjon vous demande de prouver votre valeur au combat afin de vous donner de quoi continuer votre quête auprès du Maître Épéiste de la Ville de Départ
{% endhint %}
{% endstep %}

{% step %}
#### [Retour à la Cathédrale](retour-a-la-cathedrale.md)

{% hint style="info" %}
De retour auprès du Maître Épéiste, il vous demande plus d’indices (indices de quoi idk j'ai pas lu les missions). Pour ça interrogez les Aventuriers de la Taverne de la Ville de Départ pour avoir plus de réponses
{% endhint %}
{% endstep %}

{% step %}
#### [L'Homme à Capuche](lhomme-a-capuche.md)

{% hint style="info" %}
On vous a dit d'aller en direction de Vallaht pour en savoir plus. Le maire du village vous donne alors une piste, un certain homme à la capuche. A vous de le retrouver pour en savoir plus
{% endhint %}
{% endstep %}

{% step %}
#### [Gorbel: Slime Imposant](gorbel-slime-imposant.md)

{% hint style="info" %}
Après avoir retrouvé l'homme gisant au sol, l'immense Boss Gorbel surgit de derrière les fourrées et impose sa grande masse gelatineuse. Accompagné de son armée de Slime, sa majesté peut être redoutable mais vous devez l'occire pour continuer votre quête.
{% endhint %}
{% endstep %}

{% step %}
#### [De Retour à la Cathédrale](de-retour-a-la-cathedrale.md)

{% hint style="info" %}
Le troisième fragment des anciens en votre possession, retournez voir le Maître Épéiste de la Ville de Départ. Il est temps d'assembler les 3 fragments en votre possession... votre quête toucherait-elle à sa fin ?
{% endhint %}
{% endstep %}

{% step %}
#### [Le Sceau des Anciens](le-sceau-des-anciens.md)

{% hint style="info" %}
C'est à l'Entrée du Donjon du Labyrinthe des Déchus qu'on vous apprend que celui qui pourrait vous aider est coincé à l’intérieur du Labyrinthe. À vous d'aller le sauver
{% endhint %}
{% endstep %}

{% step %}
#### [Wali](wali.md)

{% hint style="info" %}
Après avoir sauvé le jeune Wali il vous dira qui voir pour reconstruire le Sceau des Anciens. Allez retrouver son maître à Tolbana
{% endhint %}
{% endstep %}

{% step %}
#### [Méphisto](mephisto.md)

{% hint style="info" %}
Une fois Méphisto trouvé, vous allez jongler entre les discussions de Méphisto et son Apprenti à propos du Sceau des Anciens
{% endhint %}
{% endstep %}

{% step %}
#### [Wali l'Apprenti](wali-lapprenti.md)

{% hint style="info" %}
Pour continuer votre aventure, on vous emmènera voir Émy qui vous donnera les bases du parchemin que Wali vous réclame contre certaines ressources
{% endhint %}
{% endstep %}

{% step %}
#### [La Base du Parchemin](la-base-du-parchemin.md)

{% hint style="info" %}
Il est grand temps de construire la Base de Parchemin Vierge auprès d'Émy. Vous pourrez alors en apprendre plus sur les légendes de ce monde auprès de Méphisto
{% endhint %}
{% endstep %}

{% step %}
#### [Ramoon](ramoon.md)

{% hint style="info" %}
Vous avez entendu parler du mythique Nymphéa. Rendez-vous à Virelune pour en savoir plus sur cette créature sous-marine
{% endhint %}
{% endstep %}

{% step %}
#### [Chasse aux Poissons Requin](chasse-aux-poissons-requin.md)

{% hint style="info" %}
On vous a dit de voir Malrik pour en savoir plus sur le Mystérieux Serpent. Il va cependant vous demander de prouver votre valeur en affrontant des Poissons Requin pour vous aider davantage
{% endhint %}
{% endstep %}

{% step %}
#### [Nymbréa: L'ombre du Lac](nymbrea-lombre-du-lac.md)

{% hint style="info" %}
Il est grand temps d'aller exterminer Nymbréa, le Léviathan
{% endhint %}
{% endstep %}

{% step %}
#### [Sanctuaire de Xal'Zirith](sanctuaire-de-xalzirith.md)

{% hint style="info" %}
Il est grand temps de découvrir l'une des dernières zones du jeu et entrevoir votre prochain Donjon
{% endhint %}
{% endstep %}

{% step %}
#### [Retour à Tolbana](retour-a-tolbana.md)

{% hint style="info" %}
Il est grand temps de finaliser votre Quête. Pour ça vous allez utiliser tout ce que vous avez obtenu depuis le début de cette dernière et affronter 2 monstres effroyables (Sauf si vous avez eu de la Chance sur le Butin du Nymbréa)
{% endhint %}
{% endstep %}

{% step %}
#### [Le Parchemin de Sceau](le-parchemin-de-sceau.md)

{% hint style="info" %}
Le Parchemin est enfin fini, mais il faudra maintenant le droit de pouvoir l'utiliser
{% endhint %}
{% endstep %}

{% step %}
#### [Le Tombeau des Harald](le-tombeau-des-harald.md)

{% hint style="info" %}
Vous allez devoir parler aux ancestres de Tolbana pour voir si vous êtes digne d'ouvrir la porte menant au terrifiant Kobold
{% endhint %}
{% endstep %}

{% step %}
#### [Donjon: Le Kobold](donjon-le-kobold.md)

{% hint style="info" %}
Il est grand temps. Prenez votre courage à deux mains, formez une équipe de choc et venez à bout de la terreur du Palier 1 !
{% endhint %}
{% endstep %}
{% endstepper %}
