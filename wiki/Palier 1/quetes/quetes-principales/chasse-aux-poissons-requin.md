# 🦈 Chasse aux Poissons Requin

***

<h3 align="center">Difficulté ⭐⭐▪️▪️▪️</h3>

***

#### Prérequis pour commencer et rendre la mission

* Avoir terminé "[Ramoon](ramoon.md)"

#### Tâche :

{% stepper %}
{% step %}
Parlez à Malrik

{% hint style="info" %}
Malrik se trouve en plein centre de [Tolbana ](../../carte/regions/tolbana.md)(1573,1986),  à côté des marchands de cette ville
{% endhint %}
{% endstep %}

{% step %}
Tuer 10 Poissons Requin

{% hint style="info" %}
Les poissons Requin est une espèce maritime se trouvant à l'Ouest de [Virelune](../../carte/regions/virelune.md) (1477,1997)
{% endhint %}
{% endstep %}

{% step %}
Retournez voir Malrik
{% endstep %}

{% step %}
Parlez à Virel

{% hint style="info" %}
Virel se trouve à l'extrémité Ouest de [Virelune](../../carte/regions/virelune.md) (1539,1995)
{% endhint %}
{% endstep %}
{% endstepper %}

#### Récompenses

* 🧪 <mark style="color:blue;">Expérience</mark> 1100
* 🌊 [<mark style="color:blue;">Bottes de l'Écume</mark>](../../equipement/armures/sans-classe.md#bottes-de-lecume)
