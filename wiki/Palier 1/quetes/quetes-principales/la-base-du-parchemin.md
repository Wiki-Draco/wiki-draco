# 📜 La Base du Parchemin

***

<h3 align="center">Difficulté ⭐⭐⭐▪️▪️</h3>

***

#### Prérequis pour commencer et rendre la mission

* Avoir terminé "[Wali l'Apprenti](wali-lapprenti.md)"
* Pour rendre la mission il est utile d'être Mineur niveau 2

{% hint style="warning" %}
Vous pouvez acheter votre Fer en Hôtel des Ventes, ou être aidé de vos amis/guilde, si vous ne voulez pas monter votre métier de Mineur
{% endhint %}

#### Tâche :

{% stepper %}
{% step %}
Ramenez à Émy :

* 🪨 <mark style="color:$info;">Lingots de Fer</mark> 10
* 🦌 <mark style="color:orange;">Peaux de Cerf des Montagnes</mark> 16

{% hint style="info" %}
Il faut être Niveau 2 en [Mineur](../../ressources/mineur.md) pour pouvoir récolter les Minerais de Fer :

* Mine 1 (2399,3492)&#x20;
* Mine 2 (983,3475)

Les Peaux de Cerf peuvent être récupérées après avoir tué un [Cerf des Montagnes](../../monstres/carte-du-nord/tolbana/cerf-des-montagnes.md) (45% de chance)
{% endhint %}
{% endstep %}

{% step %}
Parlez à Wali puis Méphisto

{% hint style="info" %}
Wali (3195,1497) se trouve dans la Bibliothèque juste à côté de Méphisto (3203,1455)
{% endhint %}
{% endstep %}
{% endstepper %}

#### Récompenses

* 🧪 <mark style="color:blue;">Expérience</mark> 810
* 📜<mark style="color:yellow;">Parchemin Vierge</mark>
