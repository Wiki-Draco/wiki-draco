# 💀 Le Donjon des ruines maudites

***

<h3 align="center">Difficulté ⭐⭐▪️▪️▪️</h3>

***

#### Prérequis pour commencer la quête

* Avoir terminé "[Les ruines maudites](les-ruines-maudites.md)"

#### Tâche :

{% stepper %}
{% step %}
Récupérer 15 Âmes des Ruines

{% hint style="info" %}
Les Âmes des Ruines peuvent être récupérés après avoir tué chacun des [Squelettes](../../monstres/carte-du-sud/ruines-maudites/squelette-epeiste.md) des [Ruines Maudites](../../monstres/carte-du-sud/ruines-maudites/) (40% de chance)
{% endhint %}

{% hint style="warning" %}
La zone étant contestée pour les quêtes concernant la chasse de squelettes, vous pouvez acheter vos Âmes des Ruines à l'Hôtel des Ventes, ou être aidé de vos amis/guilde
{% endhint %}
{% endstep %}

{% step %}
Utiliser le Téléporteur

{% hint style="info" %}
Il vous faut 15 Âmes des Ruines pour utiliser ce téléporteur (2780,4428)
{% endhint %}
{% endstep %}
{% endstepper %}
