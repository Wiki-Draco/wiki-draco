# 🪨 Le Sceau des Anciens

***

<h3 align="center">Difficulté ⭐⭐⭐⭐▪️</h3>

***

#### Prérequis pour commencer la quête

* Avoir terminé "[De Retour à la Cathédrale](de-retour-a-la-cathedrale.md)"

#### Tâche :

{% stepper %}
{% step %}
Parlez à Catherine

{% hint style="info" %}
Catherine se trouve à l'entrée du [Donjon Labyrinthe des Déchus](../../carte/donjons/donjon-labyrinthe-des-dechus.md) (2384,2417)
{% endhint %}
{% endstep %}

{% step %}
Rejoindre les Îles Flotantes de l'[Archipel d'Ika](../../carte/regions/archipel-dika.md)

{% hint style="info" %}
Une fois téléporté à [Ika](../../carte/regions/archipel-dika.md), vous verrez un téléporteur (3278,4104) Vous aurez juste à l'utiliser comme les autres téléporteurs
{% endhint %}
{% endstep %}

{% step %}
Parlez au Spectre Mystérieux&#x20;

{% hint style="info" %}
Une fois sur les Îles Volantes de l'[Archipel d'Ika](../../carte/regions/archipel-dika.md), vous avez une petite séance de parcours afin de croiser le chemin du Spectre Mystérieux (3186,4008)
{% endhint %}

{% hint style="warning" %}
Attention la chute est bien réel, vous ne serez pas téléporter sur les Îles Volantes, vous devrez refaire le chemin. Si vous n'êtes pas un Guerrier avec de bonnes statistiques défensives vous risquerez de mourir de chute (60 ↔ 80 dégâts de chute)

De plus vous devrez être en possession des 3 Morceaux de Fragments des Anciens accumulés depuis le début de la Quête Principale
{% endhint %}
{% endstep %}

{% step %}
Reparlez à Catherine
{% endstep %}
{% endstepper %}

#### Récompenses

* 🪦 <mark style="color:$info;">Sceau des Anciens</mark>
