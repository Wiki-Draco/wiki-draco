# 🐎 Nasgul: Protecteur Maudit

***

<h3 align="center">Difficulté ⭐⭐▪️▪️▪️</h3>

***

#### Prérequis pour commencer la quête

* Avoir terminé "[Le Donjon des Ruines Maudites](le-donjon-des-ruines-maudites.md)"

#### Tâche :

{% stepper %}
{% step %}
Parler au Spectre Archiviste

{% hint style="info" %}
Coordonnées (2965,4285)
{% endhint %}
{% endstep %}

{% step %}
Récupérer 10 Tissus Maudits et 1 Cœur Putréfié

{% hint style="info" %}
Les Tissus Maudits peuvent être récupérés après avoir tué les [Squelettes Mages](../../monstres/carte-du-sud/ruines-maudites/squelette-mage.md) (50% de chance)\
Les Cœurs Putréfiés peuvent être récupérés après avoir tué le Mini-Boss Narax (70% de chance)
{% endhint %}

{% hint style="warning" %}
Les ressources demandées lors de cette quête sont disponibles uniquement dans le sous-donjon Nasgul ([Donjon Squelettes](../../carte/donjons/donjon-squelette.md))
{% endhint %}

{% hint style="danger" %}
Vous devez absolument tuer vous même les Squelettes Mages et non pas acheter les Tissus Maudits à l'HDV
{% endhint %}
{% endstep %}

{% step %}
Utiliser le Téléporteur

{% hint style="info" %}
Il vous faut les 10 Tissus Maudits et le Cœur Putréfié pour utiliser la téléporteur, et ce à chaque fois que vous voudrez le prendre
{% endhint %}
{% endstep %}

{% step %}
Vaincre Nasgul

{% hint style="info" %}
Vous voici devant le Boss du sous-donjon. Il vous faut assener le dernier coup au Nasgul, qui possède deux phases : l'une où il est sur son destrier et l'autre où il vous affronte dans un duel de lames
{% endhint %}
{% endstep %}
{% endstepper %}

#### Récompenses

* 🧪 <mark style="color:blue;">Expérience</mark> 619
* 🪦 <mark style="color:$info;">Fragment des Anciens</mark>
