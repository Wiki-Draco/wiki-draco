# Bunta

***

<h3 align="center">Difficulté ⭐⭐▪️▪️▪️</h3>

***

#### Prérequis pour commencer et rendre la mission

* Aucun prérequis pour commencer

#### Objets Demandés

* 🌳 <mark style="color:green;">Éclats de Bois Magique</mark> 5
* 🥥 <mark style="color:green;">Noyau de Slime</mark> 3
* 🍚 Poussières d'Os 5

{% hint style="info" %}
Les Éclats de bois magique peuvent être récupérés après avoir tué un [Mini Tréant](../../../monstres/carte-du-sud/hanaka/mini-treant.md) (30% de chance chacun)

Les Noyaux de Slime peuvent être récupérés après avoir tué un [Slime Soigneur](../../../monstres/carte-du-sud/vallhat/slime-soigneur.md) / [Slime Magicien](../../../monstres/carte-du-sud/vallhat/slime-magicien.md) / [Gorbel](../../../monstres/carte-du-sud/vallhat/gorbel.md) (5% de chance)

Les Poussières d'Os peuvent être récupérées après avoir tué un [Squelette Hallebardier](../../../monstres/carte-du-sud/ruines-maudites/squelette-hallbardier.md) / [Squelette Épéiste](../../../monstres/carte-du-sud/ruines-maudites/squelette-epeiste.md) (35% de chance) / [Mage Squelette](../../../monstres/carte-du-sud/ruines-maudites/squelette-mage.md) / [Tank Squelette](../../../monstres/carte-du-sud/ruines-maudites/tank-squelette.md) / [Guerrier Squelette](../../../monstres/carte-du-sud/ruines-maudites/guerrier-squelette.md) (30% de chance) / [Archer Squelette](../../../monstres/carte-du-sud/ruines-maudites/archer-squelette.md) (20% de chance)
{% endhint %}

#### Récompenses

* 🧪 <mark style="color:blue;">Expérience</mark> 293
