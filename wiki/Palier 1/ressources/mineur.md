# Mineur

{% hint style="info" %}
Il existe dans le Palier 1 trois types de minerais à récolter :

* ⚫ Charbon
* 🧱 <mark style="color:orange;">Cuivre</mark>
* 🪨  <mark style="color:$info;">Fer</mark>
{% endhint %}

<h2 align="center">Charbon</h2>

{% hint style="info" %}
Le Charbon est récupérable au niveau 1 de Mineur
{% endhint %}

{% columns %}
{% column %}
<figure><img src="../.gitbook/assets/Charbon.png" alt=""><figcaption></figcaption></figure>
{% endcolumn %}

{% column valign="middle" %}
Les Minerais de Charbon sont récupérables dans la Mine (2401,3487) au Sud du [Quartier OG](../carte/regions/quartier-og.md). Seulement dans la salle principale se trouvera environ 10 minerais.\
Une autre Mine plus grande existe (982,3474) à l'Ouest de [Hanaka](../carte/regions/hanaka.md) et du [Cyclorim](../carte/regions/cyclorim.md), avec environ 45 minerais.
{% endcolumn %}
{% endcolumns %}

***

<h2 align="center">Cuivre</h2>

{% hint style="info" %}
Le Cuivre est récupérable au niveau 1 de Mineur
{% endhint %}

{% columns %}
{% column %}
<figure><img src="../.gitbook/assets/Cuivre.png" alt=""><figcaption></figcaption></figure>
{% endcolumn %}

{% column valign="middle" %}
Les Minerais de Cuivre sont récupérables dans la Mine (2401,3487) au Sud du [Quartier OG](../carte/regions/quartier-og.md). Le long du chemin vous trouvez environ 15 minerais avant d'apparaître dans la salle principale. Après cela environ 10 minerais seront présents dans la salle principale.\
Une autre Mine plus grande existe (982,3474) à l'Ouest de [Hanaka](../carte/regions/hanaka.md) et du [Cyclorim](../carte/regions/cyclorim.md), avec environ 45 minerais.
{% endcolumn %}
{% endcolumns %}
