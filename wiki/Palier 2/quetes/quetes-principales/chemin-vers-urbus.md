# 🌇 Chemin vers Urbus

***

<h3 align="center">Difficulté ⭐⭐▪️▪️▪️</h3>

***

#### Prérequis pour commencer la quête

* Avoir terminé "[Parler à la femme étrange](parler-a-la-femme-etrange.md)"

#### Tâche :

{% stepper %}
{% step %}
Trouver Artheron à Urbus

{% hint style="info" %}
Parler à n'importe lequel des villageois de [Urbus](../../carte/regions/urbus.md) vous donnera les coordonnées précise d'Artheron (135,-374)
{% endhint %}
{% endstep %}

{% step %}
Se débarrasser des Taureaux

{% hint style="info" %}
Les [Taureaux](../../monstres/carte/lac-des-taureaux/taureau.md) sont des monstres endémique du [Lac des Taureaux](../../carte/regions/lac-des-taureaux.md), situé au Sud de [Urbus](../../carte/regions/urbus.md) (163,-79).

Il vous faudra vaincre 20 Taureaux
{% endhint %}
{% endstep %}
{% endstepper %}
