# Herboriste

{% hint style="info" %}
Il existe dans le Palier 1 deux types de Plantes à récolter :

* 🍠 <mark style="color:orange;">Sépal d'Ambre</mark>
* 🌿 <mark style="color:yellow;">Épine d'Oracile</mark>
{% endhint %}

<h2 align="center">Sépal d'Ambre</h2>

{% hint style="info" %}
Les Sépales d'Ambre sont récupérables au niveau 2 d'Herboriste
{% endhint %}
